# EduVerse — Future of Learning 🎓

A modern, premium education platform built with pure HTML, CSS, and JavaScript.

## 🌟 Features

- **🏠 Home Page** — Hero section with animated particles, feature cards, meeting previews, FAQ accordion
- **📅 Meetings** — Filterable event cards with category tabs (Workshops, Seminars, Webinars)
- **ℹ️ About Us** — Animated counter stats, mission statement
- **📚 Courses** — Searchable course catalog with ratings, pricing, instructor details
- **🛤️ Career Guidance** — Step-by-step learning roadmap with tech tags
- **📝 Resume Builder** — Live-preview resume form builder
- **📊 Dossier** — ATS score gauge, animated skill bars
- **👤 Profile** — Student information card
- **📊 Dashboard** — Full student dashboard with progress tracking, schedule, activity feed
- **🔐 Login** — Beautiful split-screen login/signup with form validation
- **🤖 EduBot** — AI chatbot assistant

## 🎨 Design

- **Colors:** Indigo (#6c5ce7) → Cyan (#00cec9) gradient with Gold (#f9ca24) accents
- **Fonts:** Outfit (headings) + Inter (body) from Google Fonts
- **Style:** Glassmorphism, gradient buttons, animated particles, smooth page transitions
- **Loading:** Branded preloader with progress bar

## 🚀 Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. No build step required — pure HTML/CSS/JS
